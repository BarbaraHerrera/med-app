# 🏥 MedApp

Aplicación móvil desarrollada con **React Native + Expo + Firebase**, que permite a los usuarios encontrar profesionales de la salud cercanos en tiempo real, visualizar su ubicación en un mapa y gestionar su perfil.

---

## 🚀 Características

- 🔐 Autenticación de usuarios (Login / Registro) con Firebase
- 🗺️ Mapa interactivo con ubicación en tiempo real
- 📍 Visualización de profesionales de la salud cercanos
- 🔎 Buscador de profesionales
- 👤 Pantalla de perfil de usuario
- 📋 Listado dinámico desde Firebase Firestore
- ☁️ Backend serverless con Firebase

---

## 🛠️ Tecnologías utilizadas

- React Native (Expo)
- Firebase Authentication
- Firebase Firestore
- React Navigation
- React Native Maps

---
## 🚀 Pitch

**MedApp** es una aplicación móvil que conecta pacientes con profesionales de la salud de forma rápida, simple y geolocalizada.

En lugar de buscar manualmente o depender de recomendaciones, MedApp permite:

- Ver profesionales cercanos en un mapa
- Filtrar por especialidad
- Acceder a información relevante en segundos

🎯 **Nuestro objetivo:** Simplificar el acceso a la salud mediante tecnología.
